## ⚙️ How to run

```
Download
Backend (Node) Folder Server

Create file at root server .env with : DATABASE_URL="file:./dev.db"

$ npm i
$ npm run dev

Frontend (React) Folder Client

$ npm i
$ npm run dev

```
